# Website UI Kit — Лычкины птички

## Overview
Hi-fidelity recreation of the Лычкины птички farm website. Built with React + Babel inline JSX.

## Screens
1. **Home (Hero + Products)** — Main landing with hero section, product grid, about strip
2. **Product page** — Detail view for a single product
3. **Order / Contact** — Simple order form

## Components
- `Header.jsx` — Fixed navigation with logo + cart
- `Hero.jsx` — Full-width hero with headline + CTA
- `ProductCard.jsx` — Individual product card
- `ProductGrid.jsx` — Grid of product cards
- `AboutStrip.jsx` — Farm story section
- `Footer.jsx` — Site footer with contacts

## Design Width
1280px desktop / responsive

## Notes
⚠️ Logo and imagery are placeholders — replace with real assets from лычкиныптички.рф
